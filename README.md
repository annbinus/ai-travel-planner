# Trippy - An AI Travel Planner

An intelligent travel itinerary generator that uses AI to create personalized multi-day trips with draggable, editable activities.

**[Live Demo](https://trippy-kappa.vercel.app)**

## Features

- **AI-Powered Generation** - Create complete itineraries from destinations and preferences
- **Geolocation** - Mapbox integration for accurate coordinates
- **Drag-and-Drop** - Reorganize activities with automatic time adjustment
- **Save & Manage** - Store multiple trip plans
- **Edit Times** - Modify activity times with time picker
- **Responsive** - Works on desktop, tablet, and mobile

## Tech Stack

- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express + PostgreSQL
- **APIs**: Mapbox for geocoding
